Aula Completa de HTML para Iniciantes

Do zero ate sua primeira pagina web

1. O que e HTML?

HTML significa HyperText Markup Language.
Ele e a linguagem usada para estruturar paginas da internet.
Pense no HTML como o esqueleto de um site: titulos, textos, imagens, links e formularios.
Sem HTML, o navegador nao sabe como organizar o conteudo.

2. Estrutura basica de um arquivo HTML

Todo arquivo HTML comeca com <!DOCTYPE html>, que informa ao navegador o tipo do documento.
Depois temos a tag <html>, que envolve toda a pagina.
Dentro dela, usamos <head> para configuracoes e <body> para o conteudo visivel.
No <head>, voce define titulo da aba, codificacao e link para CSS.

3. Tags mais importantes

<h1> ate <h6>: titulos (do maior para o menor).
<p>: paragrafo de texto.
<br>: quebra de linha.
<a>: cria links.
<img>: adiciona imagens.
<ul>, <ol> e <li>: listas.
<div> e <span>: organizacao de blocos e textos.

4. Exemplo pratico de link e imagem

Para criar um link, use <a href="URL">Texto</a>.
Para imagem, use <img src="caminho" alt="descricao">.
O atributo alt e importante para acessibilidade e SEO.
Exemplo: <img src="img/gato.jpg" alt="Foto de um gato">.

5. HTML + CSS + JavaScript

HTML: estrutura.
CSS: visual e estilo (cores, tamanhos, espacamentos, layout).
JavaScript: interatividade (cliques, animacoes, validacoes, logica).
Juntos, eles formam a base do desenvolvimento web moderno.

6. Boas praticas para iniciantes

Use nomes claros para classes e arquivos.
Sempre feche as tags corretamente.
Mantenha indentacao para facilitar leitura.
Evite exagerar no <br>; prefira <p> para separar blocos de texto.
Teste a pagina no navegador sempre que fizer alteracoes.

7. Resumo final

Se voce aprendeu o que e HTML, como funciona a estrutura basica e as principais tags,
entao ja tem uma base forte para criar paginas simples.
O proximo passo e praticar pequenos projetos e evoluir para CSS e JavaScript.
Programacao web e pratica constante: quanto mais voce testa, mais rapido aprende.
